== Description ==

CryptoPulse Whale Alerts brings real-time whale wallet tracking to your WordPress site. Monitor large crypto transactions across 34+ EVM chains including Ethereum, BSC, Polygon, Arbitrum, and more.

**Features:**
* Real-time whale movement tracking
* 34+ EVM chains supported
* Dark and light themes
* Auto-refresh every 60 seconds
* Shortcodes and sidebar widget
* Single wallet lookup
* AI-powered wallet roasting

**Shortcodes:**
* `[cryptopulse_whales chain="ethereum" limit="10" theme="dark"]`
* `[cryptopulse_wallet address="0x..." theme="dark"]`

== Installation ==

1. Upload `cryptopulse-widget` to `/wp-content/plugins/`
2. Activate the plugin
3. Go to Settings → CryptoPulse
4. Enter your API key from [cryptopulse.uno](https://cryptopulse.uno)
5. Use shortcodes or the widget
